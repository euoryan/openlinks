# Open Links

Este é um projeto em Python que serve como base para criar um executável que abre links pré-definidos no código. A ferramenta é ideal para compartilhar com clientes ou usuários, facilitando o acesso rápido a links importantes.

## Sobre
O projeto permite que você crie um executável simples que abre links específicos diretamente no navegador, sem a necessidade de digitar ou buscar pelos links manualmente. Isso pode ser útil para campanhas, promoções ou qualquer situação onde você queira direcionar os usuários para URLs pré-determinados.

## Tecnologias
- Python

## Como usar
1. Clone o repositório:
   ```bash
   git clone https://github.com/euoryan/openlinks
   ```
2. Acesse o diretório do projeto:
   ```bash
   cd openlinks
   ```
3. Instale as dependências necessárias:
   ```bash
   pip install -r requirements.txt
   ```
4. Execute o script para abrir os links:
   ```bash
   python openlinks.py
   ```
